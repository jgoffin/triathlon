Predict Your Triathlon Swim Time (Olympic Distance: 1500m)

When I was training for my first triathlon, I had no previous experience swimming competitively. But I wanted to know what kind of time I should be aiming for! So, I was curious: could I build a machine learning model to predict a swim time based on my age, gender, and expected bike/run times? The resulting project involved scraping the web for Olympic Triathlon race results, building a model, and deploying it on this website. Enter some values, and see your prediction! If you're curious about the web-scraping code or ML portion, find it all on my repo here.
